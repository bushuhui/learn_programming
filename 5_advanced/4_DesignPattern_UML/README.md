# 设计模式与UML


## 设计模式
设计模式是软件设计中常见问题的典型解决方案。 每个模式就像一张蓝图，你可以通过对其进行定制来解决代码中的特定设计问题。设计模式与方法或库的使用方式不同，你很难直接在自己的程序中套用某个设计模式。模式并不是一段特定的代码，而是解决特定问题的一般性概念。你可以根据模式来实现符合自己程序实际所需的解决方案。
